# Code of Conduct

We're a small project. The rules are the same as anywhere else in life: be decent, don't be a jerk, assume good intent.

## Specifically

- Be respectful in issues, PRs, and discussions.
- Disagreements about engineering are great. Personal attacks are not.
- If you're new to open source, ask questions. There are no bad questions in this repo.
- If someone files an issue describing a real production problem, treat it as gold. Help them, don't lecture them.

## Not OK

Harassment, discrimination, doxxing, deliberate trolling, threats of violence, sexualized content, or any conduct that would make a reasonable person feel unsafe.
